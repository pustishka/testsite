Pet-Projects

  1. Steam-invest Checker – a program for automatically calculating the value of Steam inventory, allows you to fill in data about the existing inventory of Steam using the GUI interface, take data from a file. Parses prices from the Steam community site, displays in the interface the sum of each item + the sum of all items (writing the result with the date in the file). Current profit or loss based on the previous calculation. (Python, Tkinter, Beautiful Soup4, Requests, Threading)

  2. PoorSales – web project on Django, web service for placing discounts on cafes, fast food restaurants, shops. Implemented a breakdown of discounts by category, adding discounts, registration, authorization, editing a personal profile, displaying discounts depending on the selected favorite category when registering a profile. Comment system with date and author. Automatic mailing of new discounts to specified email addresses of users. Successfully deployed to the server heroku, and added in docker container. All data in SQL (Python, Django + ORM, SQlite, Html, CSS, Docker)

  3. HowFarDidYouRun – telegram bot on aiogram, is a multiplayer game between bot users, based on the calculation of the distance traveled between two geolocation points, top players, leaderboard, saving individual information in the database, with deploy on railway (Python, aiogram, asyncio, harvesine, sqlalchemy)

  4. Tic-tac-toe – ready-made Tic-Tac-Toe game client for two players (offline), with a completely ready-made GUI in Tkinter with a main menu and a choice of names for players. (Python, Tkinter, Threading)
